# 🌙 Moon Management App

A sleek, dark-themed Streamlit application to keep thorough tabs on the moon. Track lunar phases, astronomical events, and daily astrological "Moon Vibes."

## ✨ Features

- **Real-time Lunar Tracking**: Track phase name, emoji, illumination percentage, and moon age.
- **Full Moon Countdown**: Always know exactly when the next full moon is coming.
- **Moon Vibes**: Get daily astrological insights based on the moon's current zodiac transiting sign.
- **2026 Highlights**: Integrated calendar for 2026 eclipses, supermoons, and meteor showers.
- **Scheduler-Friendly**: Includes `check_full_moon.py` for automated alerts (perfect for Telegram/Slack integrations).

## 🚀 Quick Start

1. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the app**:
   ```bash
   streamlit run app.py
   ```

## 🛠️ Tech Stack

- **Python**: Core logic.
- **Streamlit**: Modern UI/UX.
- **PyEphem**: High-precision astronomical calculations (accurate to ±1 minute).
