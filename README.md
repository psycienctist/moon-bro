# 🌑 Lunatick

A high-performance, cosmic-themed Streamlit application for tracking the moon's position, phases, and astrological influence.

## 🚀 Features

- **Primary Countdown**: The next Full Moon countdown is front and center.
- **Real-time Lunar Tracking**: Track phase, illumination, and cycle age.
- **Moon Vibes**: Astrological insights based on the moon's zodiac transit.
- **2026 Cosmic Calendar**: Major eclipses and astronomical events.
- **Automated Alerts**: Compatible with Telegram/Slack schedulers via `check_full_moon.py`.

## 🛠️ Tech Stack

- **Python**: Core logic.
- **Streamlit**: Modern UI with a custom cosmic CSS theme.
- **PyEphem**: High-precision astronomical calculations.
